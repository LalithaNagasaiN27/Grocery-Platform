STEPS TO RUN THE APPLICATION :

1. INSTALL PHP,PHPMYADMIN, XAMPP and VSCODE
2. Place the file in 'htdocs' inside the xampp folder
3. Start APACHE and MYSQL in XAMPP
4. OPEN The webpage in any browser(chrome, firefox, and edge) using localhost server and then directory of the application (ex: localhost:3000/grocerystore/grocerystore)
